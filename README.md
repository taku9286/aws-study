・発生したエラー
2026年4月3日、cloudformationでEC2作成中にロールバック発生。エラーログの追い方とGitHubへの初プッシュを学んだ。
・確認したログ
画像のタイムラインで赤くなっている部分（根本原因）を確認。
・原因
エラー理由に 「not eligible for Free Tier（無料枠の対象外です）」 と書いてあった。これは、AWSアカウントが現在「無料枠（Free Tier）」を使える状態にない（あるいは期限が切れている）のに、無料枠前提の設定でリクエストを送ったためにAWS側で拒否された、という状況だった。
・解決策
無料枠で行っていたため、t2.micro を指定していたがこれを t3.micro （最新世代で安定しているタイプ）に変更。成功後、スタックをそのままにしていると追加でお金がかかってしまうため、スタックを削除した。
