# 🌐 AWS Custom VPC & Web Server Project

AWS上に独自の仮想ネットワーク（VPC）を設計・構築し、その中でWebサーバーを稼働させることに成功しました。

## 🏗️ システム構成
- **VPC**: 10.0.0.0/16
- **Subnet**: 10.0.1.0/24 (Public)
- **EC2**: t3.micro (Amazon Linux 2023)
- **Middleware**: Apache (httpd)

## 🛠️ トラブルシューティング（学び）

### 1. 接続タイムアウトの解決
**【現象】**
パブリックIPをブラウザに打っても「応答時間が長すぎます」と表示される。
**【原因】**
ブラウザが自動的に `https://` (Port 443) で接続しようとしていたが、セキュリティグループでは `http://` (Port 80) しか許可していなかった。
**【解決】**
URLの先頭に明示的に `http://[IPアドレス]` と入力することで疎通確認。

### 2. CloudFormationのパースエラー
**【原因】**
YAMLファイルのインデント（字下げ）や、コピー＆ペースト時の不要な空白。
**【解決】**
エディタ（VS Code等）を使用してフォーマットを整え、まっさらな状態から「スタックの新規作成」を行うことで解消。
